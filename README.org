* Moodl-rs



To init the db and courses:

#+begin_src sh
cargo run -- --wstoken <wstoken>
#+end_src

To fetch grades for a given course:

#+begin_src sh
cargo run -- --wstoken <wstoken> --courseid <courseid>
#+end_src

Replace ~<wstoken>~ with your "Moodle Mobile additional features service" key.
